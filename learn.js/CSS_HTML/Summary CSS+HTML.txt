    Блочный тег занимает всю ширину экрана. Начинается сн новой строки.
        <header> - верхняя часть сайта (блочный тег)
        <nav> - для навигации или меню (блочный тег)
        <footer> - нижняя часть сайта (блочный тег)
        <div> - блочный тег

    Заголовки
        <h1> - только один на страницу! (блочный тег)
        <h2>...<h6> - многоразовые. (блочный тег)
    Абзац
        <p> -  (блочный тег)
        <span> -  (строчный тег) ширина зависит от ширины содержимого!
                Используется для стилизации текста(части текста)
    Cсылки

            <a href="yandex.ru">Text</a>
        <a target="_blank" href="yandex.ru"/>Text</a> - ссылка откроется в новом окне
    Картинка
        <img src="путь" alt="альтернативный текст"/>
    Списки(блочный)
        <ul> - маркированный список
        <ol> - нумерованный список
        <li> - элемент списка
    
    Перевод строки <br />

    <form action="#"></form> - блочный, для передачи значений на сервер
        - action - адрес прораммы, которая будет обрабатывать переданную информацию
        - method - get/post
            - get - передача идет прямо в адресной строке и могут быть использованы неоднократно(видно всем)
            - post - передает данные скрытно
            - enctype - используется совместно с post

        <form action="#" method="post" enctype="multipart/form-data"></form>
    
    <input type="text" name="" value=""/> строчный 
        - type - тип поля
            text - текстовое поле
            password - поле с паролем
            radio - переключатель
            checkbox - флажок
            hidden - скрытое поле
            button - кнопка
            submit - кнопка для отправки формы
            reset - кнопка для сброса формы
            file - поле для отправки файла
            image - кнопка с изображением
        placeholder - подпись подсказка

    <textarea> - (строчный) многострочное текстовое поле
        <textarea name="expanded" id="" cols="30" rows="10"></textarea>

    ОБНУЛЯЮЩИЙ СТИЛЬ
        * {
            padding: 0;
            margin: 0;
            border: 0;
        }

        /* Меняет подсчет ширины блока */
        *,*::before, *:after{
            -moz-box-sizing: border-box;
            -webkit-box-sizing: border-box;
            box-sizing: border-box;
        }
        /* Линии обводки у браузеров */
        :focus, :active{outline: none;}
        a:focus, a:active{outline: none;}
        /* Делаем блочными теги: */
        nav,footer,header,aside{display: block;}
        /* Уравниваем поведение шрифтов и элементов для разных браузеров(например, запрет на самовольное увеличение шрифта на IPhone) */
        html,body{
            height: 100%;
            width: 100%;
            font-size: 100%;
            line-height: 1;
            font-size: 14px;
            -ms-text-size-adjust: 100%;
            -moz-text-size-adjust: 100%;
            -webkit-text-size-adjust: 100%;
        }
        /* Наследование шрифтов*/
        input,button, textarea{font-family: inherit;}
        /* Убирается крестик в input */
        input::-ms-clear{display: none;}
        /*Убираются отступы кнопок в разных браузерах*/
        button{cursor: pointer;}
        button::-moz-focus-inner {padding: 0;border:0;}
        /*Чистятся стили маркированных списков*/
        a, a:visited{text-decoration: none;}
        a:hover{text-decoration: none;}
        ul li{list-style: none;}
        img{vertical-align: top;}
        /*Обнуляем все заголовки*/
        h1,h2,h3,h4,h5,h6{font-size: inherit;font-weight: 400;}

    ШРИФТЫ
        Property(snippet)
        font-family(ffa) - смена шрифта 
            font-family: Arial, "Helvetica Neue", Helvetica, sans-serif;
        font-size(fz) - размер шрифта
            font-size: 30px;
        font-style(fs) - стиль текста
            font-style: italic;
                - italic - курсивный(имитация рукописного)
                - oblique - наклонный
                - normal - обычный
        font-weight(fw) - толщина(вес) шрифта
        color() - цвет
        text-align(tal/tac/tar/taj) - выравнивание
            - left
            - center
            - right
            - justify - растягивать на всю ширину.
        text-decoration(td) - подчеркивание/зачеркивание/линия сверху
            - underlie
            - overline
            - line-through - зачеркивание
        text-shadow - тень текста
            text-shadow: 1px 1px 1px #000;
        text-transform - заглавные/прописные/каждое слово с большой
        text-ident(ti) - отступ первой строки
        letter-spacing(ls) - расстояние между символами
        word-spacing() - расстояние между словами
        white-space() - перенос слов
            - normal
            - nowrap - запрет переноса слов
        line-height(lh) - расстояние между строк

    Отступы
        padding - отступ внутри(только для блочных)
            не позволяет отрицательные значения
            padding: 10px 20px 30px 40px;
                сверху/справа/снизу/слева
            padding: 10px 20px 30px;
                сверху/справа-слева/снизу
            padding: 10px 20px;
                сверху-снизу/справа-слева
            padding: 10px;
                одно со всех сторон

        margin - внушний отступ блочных элементов
         возможны отрицательные значения
            margin: 10px 10px 10px 10px;

        width - ширина блока
            в основном для блочных тегов/картинок
        
        max-width - максимальная ширина(для блочных)
        min-width - минимальная ширина(для блочных)
        height - высота блока
        min-height/max-height - мин/макс. высота блока

        box-sizing: border-box
         по умолчанию width и height не включают в себя размер отступов и границы
         - border-box - включают размеры отступов и границ, но не margin

        overflow - управляет отображением содержания блочного элемента
            - visible - показывает содержимое блока всегда
            - hidden - скрывает все, что не убирается в границы блока.
            - scroll - прокрутка
            - auto - подключает прокрутку только если необходимо

        display
            -block - заставляет вести строчные теги как блочные
            -inline - заставляет вести тег как строчный
            -inline-block - остается строкой, но действуют ширина/высота/отступы
            -none - временно удаляет элемент со страницы
