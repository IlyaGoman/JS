1. HTML
    <!DOCTYPE html>
    <html lang="ru">
        <!--Заголовок страницы, контейнер для других данных, не отображается-->
        <head>
            <titl>First page!</title>
            <meta http-equiv="Content-type" content="text/html;charset=UTF-8"/>
        </head>
        <!--Тело страницы, все что видно-->
        <body>
            <h1>Test project!</h1>
        </body>

    Блочный тег занимает все предоставленное пространство
        <header> - верхняя часть сайта (блочный тег)
        <nav> - для навигации или меню (блочный тег)
        <footer> - нижняя часть сайта (блочный тег)
        <div> - блочный тег
